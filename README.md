# 🏠 家庭库存管理助手

一个**纯前端、移动端优先**的家庭物品库存管理网页应用。  
无需安装 App，打开浏览器即可使用，支持添加到手机主屏幕，体验接近原生 App。

---

## ✨ 主要功能

- 📝 **快速录入**：物品名称、分类（必需品/等优惠再买/数码类/其他）、数量、最低库存阈值、优先级
- 📊 **可视化看板**：饼图展示分类占比，柱状图展示库存状态（正常/需补货/已用完），颜色区分一目了然
- ⚠️ **急需购买清单**：自动汇总需补货物品，按优先级排序，红色徽标提醒数量
- 🔍 **灵活筛选**：按分类、状态筛选，按优先级排序，一键 +1/-1 调整库存
- 💾 **数据备份**：导出 JSON 文件备份，支持导入恢复，数据存储在浏览器本地，不经过任何服务器
- 📱 **App 式体验**：响应式设计，添加到手机主屏幕后全屏运行，可离线使用（需 PWA 支持）

---

## 🚀 如何使用

### 方式一：在线使用（推荐）
1. 将项目部署到静态托管服务（如 Gitee Pages、GitHub Pages）
2. 用手机浏览器打开对应网址
3. 点击浏览器的「添加到主屏幕」（或「添加到桌面」）
4. 桌面生成图标，之后像 App 一样打开即可

### 方式二：本地使用
1. 下载 `index.html` 文件
2. 用手机文件管理器找到文件，选择用浏览器打开
3. 同样可以「添加到主屏幕」（部分浏览器对本地文件支持有限，建议托管使用）

---

## 🛠️ 技术栈

- 纯 HTML/CSS/JavaScript，单文件
- [Chart.js](https://www.chartjs.org/) 驱动图表
- 数据通过 `localStorage` 本地存储
- 可选 PWA 支持离线使用

## 📱 添加到主屏幕步骤
